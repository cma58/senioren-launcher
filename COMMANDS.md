# 📟 Full Overview of SMS Commands

This page contains all available commands to manage the Senior Launcher remotely. Send these codes via a standard SMS from your own phone to the senior's phone.

> **Note:** Commands only work if your number is registered as an **SOS contact** in the senior's app.

---

## 🛡️ 1. Safety & Location
These commands are crucial for the senior's safety.

| Command | Action | Situation |
|:---|:---|:---|
| **`#WAAR`** | Sends back a Google Maps link with the current GPS location. | Use this if the senior is lost or the phone is lost outside. |
| **`#PING`** | Shows a large "ARE YOU OKAY?" button on the screen. | For a quick wellness check without disturbing them with a call. |
| **`#BEL_TERUG`** | Makes the senior's phone call your number immediately. | Useful if the senior doesn't know how to call or is in panic. |
| **`#SOS_NU`** | Directly starts the full emergency procedure (SMS + calling all SOS contacts). | In case of an acute emergency where the senior cannot press the button. |
| **`#VEILIG ON`** | Blocks all incoming calls from numbers not in the contacts. | If the senior is frequently harassed by telemarketers or scammers. |
| **`#BLOKKEER 06...`**| Adds a specific number to the blacklist. | To permanently block a specific unwanted number. |

---

## ⚙️ 2. System & Connectivity
Restore settings the senior may have changed by accident.

| Command | Action | Situation |
|:---|:---|:---|
| **`#WIFI ON`** | Enables the WiFi receiver. | If the senior accidentally turned off WiFi and is using unnecessary mobile data. |
| **`#STIL OFF`** | Disables "Do Not Disturb" and restores normal sound. | If the senior doesn't hear calls because the device is on silent. |
| **`#HELDER 10`** | Sets screen brightness to maximum (scale 1-10). | If the senior cannot see the screen because it's too dark. |
| **`#VOLUME 10`** | Sets ringtone to maximum volume (scale 0-10). | If the senior repeatedly misses calls. |
| **`#BT ON`** | Enables Bluetooth. | For automatic pairing of hearing aids or an SOS bracelet. |
| **`#RESTART`** | Restarts the launcher app. | If the app behaves strangely or hangs. |

---

## 🔦 3. Search & Tools
Help finding the phone or providing extra light.

| Command | Action | Situation |
|:---|:---|:---|
| **`LAUN_ZOEK`** | The phone shouts: "I am here!" and vibrates. | To find the phone under the couch or in another room. |
| **`#LAMP ON`** | Turns the flashlight on constantly. | If the senior needs to find something in the dark. |
| **`#LAMP_AUTO 5`**| Turns the flashlight on for 5 minutes and then off automatically. | Prevents battery drain if the senior forgets to turn it off. |
| **`#KNIPPER`** | Makes the flashlight blink 10 times. | A visual signal to find the phone without too much noise. |

---

## 📱 4. UI & App Management
Adjust the interface to the senior's needs.

| Command | Action | Situation |
|:---|:---|:---|
| **`#LETTER 4`** | Sets all texts in the app to very large (level 1-5). | If the senior has trouble reading the screen. |
| **`#THEMA 2`** | Switches to the "High Contrast" theme (yellow on black). | For seniors with visual impairments who need more contrast. |
| **`#SLOT ON`** | Locks the settings button for the senior. | Prevents the senior from changing settings and getting lost. |
| **`#OPEN Photos`**| Directly opens the Photos app in the foreground. | "I can't find the pictures of the grandkids anymore!" |
| **`#NOTIFICATIES_WEG`**| Clears all notifications at the top of the screen. | Cleans up the screen if there are too many (confusing) alerts. |

---

## 🗓️ 5. Planning & Medication
Manage the calendar and health remotely.

| Command | Action | Situation |
|:---|:---|:---|
| **`#MEDICIJN 08:00 Aspirin`** | Adds a daily medication reminder. | Manage medication use without being physically present. |
| **`#VOORRAAD Aspirin 20`**| Updates the number of remaining pills. | After you have picked up new medication for the senior. |
| **`#WEKKER 07:30 Wake up`** | Sets a one-time or daily alarm. | Helps the senior remember an early morning appointment. |
| **`#AGENDA_VANDAAG`** | You receive an overview of today's appointments. | To check if the senior has important meetings. |

---

## 🔍 6. Diagnostics & Status
Get insight into the device's state.

| Command | Action |
|:---|:---|
| **`#STATUS`** | Shows battery %, current volume, and silent mode status. |
| **`#INFO_PLUS`** | Shows free storage space, Android version, and signal strength. |
| **`#NETWERK`** | Shows the provider (e.g., Vodafone) and connection type (4G/5G). |
| **`#LAATSTE_OPROEP`**| Shows who called last and when. |
| **`#APP_LIJST`** | Gives a list of the most important apps on the phone. |

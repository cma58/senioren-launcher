## 🛡️ Security & Privacy (EU & Android 16 Ready)
This project is updated to comply with the latest European security standards (**GDPR/AVG**) and is fully optimized for **Android 16 (Baklava)**:
- **Privacy by Design:** Explicit privacy labels for sensitive data (Contacts, GPS).
- **Secure Updates:** Update delivery via encrypted HTTPS connections.
- **Modern Display:** Full Edge-to-Edge support for Android 15/16.
- **100% Local:** Your data never leaves your device. No cloud, no tracking.

## 🌟 My Vision: "Senior-First"
Most smartphones are designed for young people. We turn that around. The Senior Launcher is built following strict UX rules for the elderly:
- **No Keyboards:** Everything works with large Plus/Minus buttons and simple lists.
- **Gigantic Elements:** Texts are at least 20-30sp. Buttons are at least 70dp high.
- **Digital Peace:** No unnecessary notifications or complicated swipe gestures.

## ✨ Key Features
- **📞 Calling:** Large dialer with photo favorites and 112 emergency confirmation.
- **💬 Messaging:** Extra-large SMS view with instant font size adjustment.
- **🆘 SOS:** Hold for 3 seconds for emergency calls and GPS location sharing.
- **💊 Medication:** Reminders with actual pill photos and stock tracking.
- **🎮 Remote Support:** Manage the phone via simple SMS codes (no internet required).
